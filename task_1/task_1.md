# Задание

Найдите баги на приложенном скриншоте, приоритезируйте их и перечислите списком. Приоритеты: P0 — критический, P1 — высокий, P2 — средний, P3 — низкий.

# Найденные баги

- P0 - Раздел "Подобрали для вас" содержит объявления не по запросу пользователя (должны быть персидские кошки)
- P0 - Неверная вложенность навигации: вместо "Главная - Животные - Персидская - Кошки" должно быть "Главная - Животные - Кошки - Персидская"
- P0 - Неверный тоггл "Сначала из Можайска" при выбранном городе Москва
- P1 - Строка "Ошибка, обновите страницу" не должна показываться пользователю
- P2 - Первая карточка с локацией в Новосибирске, хотя выбран город Москва
- P2 - Объявление с плюшевым котенком не подходит под запрос пользователя, т.к. это не животное
- P2 - Карточки объявлений во втором ряду расположены криво
- P2 - Неверная подсказка "Не показывайте мне похожие объявления" при наведении на кнопку "Избранное"
- P2 - Есть объявление "Экзотический короткошерстный котенок" при выборе персидской породы
- P3 - Опечатка в кнопке "Корьера в Авито"
- P3 - Выбор возраста "Щенок" при просмотре объявлений кошек не должен отображаться
- P3 - Заголовок "Котята в Москве" не изменился после выбора породы

Это баги для следующего состояния страницы: выбрана порода кошек и показываются объявления по ней, остальные фильтры указаны, но кнопка "Показать 8 объявлений" не нажата

Если фильтры настроены и нажата кнопка "Показать 8 объявлений", то можно выделить такие баги:

- P2 - Рядом с надписью "Котята в Москве" должно быть актуально количество объявлений, то есть 8 вместо 19
- P2 - Не должно быть объявления "Персидский котенок девочка", потому что выбран другой пол
- P2 - Не должно быть объявления с котенком с ценой 500 Р, потому что установлен фильтр цены
- P2 - Не должно быть объявления с котенком с ценой 40 000 Р, потому что установлен фильтр цены
