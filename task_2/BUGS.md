## BUG-1: Фильтр «Диапазон цен» не отфильтровывает все объявления

**Номер:** BUG-1  
**Окружение:** Десктоп, Chrome, `https://cerulean-praline-8e5aa6.netlify.app`

**Шаги:**

1. Открыть страницу «Модерация Авито» (список объявлений).
2. В фильтре «Диапазон цен (₽)» задать минимум 10000 и максимум 15000.
3. Дождаться обновления списка объявлений.

**Ожидаемый результат:**  
Все отображаемые объявления имеют цену в диапазоне 10000–15000 ₽.

**Фактический результат:**  
В выдаче присутствует объявление с ценой 16683 ₽ (выходит за пределы заданного диапазона).

## BUG-2: Автообновление статистики не выключается при нажатии на паузу

**Номер:** BUG-2  
**Окружение:** Десктоп, Chrome, `/stats`, `https://cerulean-praline-8e5aa6.netlify.app`

**Шаги:**

1. Открыть страницу «Статистика» модератора.
2. Убедиться, что автообновление включено (таймер «Обновление через» уменьшается).
3. Нажать кнопку управления автообновлением («Пауза»).
4. Подождать несколько секунд.

**Ожидаемый результат:**  
Автообновление отключено: значение таймера перестаёт меняться, состояние автообновления — «выключено».

**Фактический результат:**  
Таймер продолжает изменяться, автообновление фактически не выключается.

## BUG-3: Тоггл «Только срочные» не фильтрует все объявления

**Номер:** BUG-3  
**Окружение:** Десктоп, Chrome, `https://cerulean-praline-8e5aa6.netlify.app`

**Шаги:**

1. Открыть страницу «Модерация Авито» (список объявлений).
2. Включить тоггл «Только срочные».
3. Дождаться обновления списка объявлений.

**Ожидаемый результат:**  
В списке остаются только срочные объявления; у каждого объявления отображается метка «Срочно».

**Фактический результат:**  
Часть объявлений отображается без метки «Срочно».
