# Тест-кейсы

## Десктоп версия

**1. Проверка фильтра "Диапазон цен" (есть баг)**

_Шаги:_

- Открыть главную страницу "Модерация Авито"
- Прокрутить страницу до фильтра "Диапазон цен"
  - Фильтр имеет название "Диапазон цен (Р)"
  - Фильтр имеет два активных поля ввода минимальной и максимальной цены
- В поле минимальной и максимальной цен ввести суммы 10000 и 15000
  - На странице отображаются только объявления, цена которых находится в диапазоне от 10000 до 15000

**2. Проверка сортировки «По цене»**

_Шаги:_

- Открыть главную страницу "Модерация Авито"
- В фильтре "Сортировать по" выбрать значение "По цене"
  - Объявления на странице отсортированы по цене в порядке убывания

**3. Проверка фильтра «Категория»**

_Шаги:_

- Открыть главную страницу "Модерация Авито"
- В фильтре "Категория" выбрать первую категорию из списка
  - На странице отображаются только объявления выбранной категории

**4. Проверка тоггла «Только срочные» (есть баг)**

_Шаги:_

- Открыть главную страницу "Модерация Авито"
- Включить тоггл "Только срочные"
  - У всех объявлений на странице отображается метка "Срочно"
  - Значение фильтра "Приоритет" изменяется "Срочный"

**5. Проверка работы контейнера управления таймером обновления статистики.**

_Шаги:_

- Открыть главную страницу "Модерация Авито"
- Выбрать раздел "Статистика"
  - Открылась страница Статистики
- Нажать кнопку "Обновить"
  - Таймер заново начал отсчет
- Нажать кнопку "Пауза"
  - Таймер остановил отсчет
  - Появилась надпись "Автообновление выключено"
- Нажать кнопку "Продолжить"
  - Таймер продолжил отсчет
  - Пропала надпись "Автообновление выключено"

## Мобильная версия

**6. Проверка переключения между темной и светлой темами**

_Шаги:_

- Открыть главную страницу "Модерация Авито"
- Нажать на кнопку изменения темы
  - Тема изменилась
